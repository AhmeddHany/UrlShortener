# UrlShortener
High-performance URL Shortener API built with .NET 9, optimized with Redis caching and PostgreSQL/SQL Server, fully containerized using Docker Compose.

Caching Strategy: Using Redis to reduce DB load and speed up redirection.

Rate Limiting: Protects the API from brute-force/spamming.

Containerization: Easy to run with a single docker-compose up command.

Stress Tested: Capable of handling bulk URL generation.
